📋 PARA USAR EN PS3:
─────────────────────────────────────────────────────────────────────────────
   1. Transferir hello-ps3.pkg a la PS3 (USB/FTP/HTTP)
   2. Instalar desde Package Manager
   3. Ejecutar "Hola Mundo PS3" desde el XMB
   4. Presionar X en el control para salir

💻 COMPILACIÓN:
─────────────────────────────────────────────────────────────────────────────

   make pkg       # Crear PKG instalable
